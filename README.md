- Vanilla JavaScript (ES6+)

## 実行方法

各フォルダ内の `index.html` をブラウザで開くだけ！

```bash
# ローカルサーバーを使う場合
cd day001-signup
python -m http.server 8080
# または
npx serve
```

## 学習ポイント

### Day 001 - Sign Up
- CSS Custom Properties でテーマカラー管理
- パスワード強度インジケーター
- フォームバリデーション（リアルタイム）
- スムーズなアニメーション（keyframes）
- アクセシビリティ考慮（aria-label）

### Day 002 - Credit Card Checkout
- リアルタイムカードプレビュー
- カードタイプ自動検出（Visa, Mastercard, Amex等）
- Luhnアルゴリズムによる番号検証
- 自動フォーマット（4桁区切り、MM/YY）
- カードタイプ別グラデーション
- レスポンシブデザイン

### Day 003 - Landing Page
- AI Assistant "Potato AI" をテーマにしたランディングページ

- グラデーションテキスト効果
- Intersection Observer でスクロールアニメーション
- チャットウィンドウのタイピングアニメーション
- フローティングカードのパララックス効果
- 統計カウンターアニメーション
- 完全レスポンシブデザイン（モバイル対応）

### Day 004 - Calculator

- 演算子ハイライト状態
